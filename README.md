# Task 02: Number Guessing Game

## 📌 Project Overview
This project is a functional **Number Guessing Game** developed as part of my internship at **SkillCraft Technology**. The program generates a random number and challenges the user to guess it, providing real-time feedback on whether their guess is too high or too low.

### Objectives:
* Implement random number generation.
* Manage user input and handle data validation.
* Use conditional logic (`if-else`) and loops (`while`) for game flow.

---

## 🚀 Features
* **Randomized Gameplay:** Every session generates a new secret number.
* **Smart Feedback:** Tells the user if their guess is "Too High," "Too Low," or "Correct."
* **Attempt Counter:** Tracks how many tries it takes to win.
* **Error Handling:** Built-in protection against invalid (non-numeric) inputs.

---

## 🛠️ Built With
* **Language:** Python 3.x
* **Library:** `random` (Standard Library)
